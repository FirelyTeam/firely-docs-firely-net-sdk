# Release notes

The release notes can be found in the [Firely SDK repository](https://github.com/FirelyTeam/firely-net-sdk/releases).

## What's new in SDK6?

If you are familiar with our previous versions of the SDK, you might still want to take some time to familiarize yourself with the changes and new features in SDK6.

Some of the key changes include:

* Improved support for [dynamic access to FHIR data](model/dynamic-features), including new methods for working with overflow data and invalid values.
* Enhanced [equality comparison features](model/equality-and-cloning), allowing for more customization when comparing FHIR model instances.
